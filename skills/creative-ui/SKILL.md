# Creative UI

> Build distinctive, memorable UIs — reject templates, find character, break conventions thoughtfully.

## When to use

Use this skill when you want a UI that stands out — not another SaaS template. When you're tired of shadcn defaults, purple gradients, and cookie-cutter components.

## Core philosophy

### Reject the default
Before reaching for component libraries, ask: does this need to be a generic dropdown/tabs/modal? Can the interaction be simpler, more playful, more bespoke? Build custom components when the experience benefits from it.

### Design with intent
Every color, spacing choice, typeface, and animation should have a reason. If you can't explain why something looks the way it does, it's decoration, not design.

### Find the character
Every product has a personality. A developer tool should look like a sharp instrument. A creative tool should feel inspiring. A fintech app should feel trustworthy. Go beyond "clean SaaS."

### Break the grid thoughtfully
Asymmetric layouts, unexpected white space, broken alignment, and intentional tension make a UI memorable — when done with purpose.

### Color beyond purple
Explore unusual palettes: muted earth tones, high-contrast monochrome with one accent, clashing colors that work, or type-only interfaces. Pick palettes that reflect the product's character.

### Typography as identity
Don't default to Inter or system-ui. Pick typefaces with personality. Pair them intentionally. Use typographic scale as a design tool.

## Design principles

1. **Start with content, not chrome** — build around what matters, then add surfaces
2. **Less is more, but not boring** — minimalism without personality is empty
3. **Motion with meaning** — animate to guide attention, show relationships, or delight
4. **Dark mode is not just inverted colors** — design two distinct experiences
5. **Responsive is not mobile-first-thin** — think about how the experience changes
6. **Empty states are prime real estate** — make them count
7. **Micro-interactions > macro-animations** — the feel of a toggle, the snap of a scroll

## Inspiration sources

- Brutalist web design
- Print editorial layouts
- Swiss design / International Typographic Style
- Japanese minimalist design
- Memphis Group / postmodern
- Retro computing / terminal UIs
- Zine aesthetics
- Museum and gallery exhibition design
- Wayfinding and signage systems
- Gaming UI (clarity and feedback, not skeuomorphic)

## Avoid referencing

- Tailwind UI component examples
- Shadcn/ui default styles
- Material Design, Ant Design, Bootstrap
- "Modern SaaS landing page" patterns
